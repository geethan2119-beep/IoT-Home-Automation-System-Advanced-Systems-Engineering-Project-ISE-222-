# IoT-Home-Automation-System-Advanced-Systems-Engineering-Project-ISE-222-


Designed a unified IoT smart home hub that integrates cameras, HVAC, lighting, doors, energy meters, and smoke detectors through multi-protocol wireless connectivity and a single mobile app.
Conducted a full customer-needs analysis and validation; identified key priorities such as seamless device integration, security, remote access, and UI convenience, using surveys and data-driven evaluation.
Developed end-to-end system architecture including context diagrams, interface definitions, operational requirements, TPM metrics, House of Quality, and lifecycle costing.
Defined system boundaries, security processes, maintenance workflows, and scalability requirements to ensure a robust, reliable, and user-friendly smart home ecosystem.
